# Claude Fable 5 Enhancer JS

A small Node.js prompt enhancer for long-form fiction planning, chapter drafting,
scene cleanup, and revision-friendly Claude Fable workflows.

The script takes a rough story idea and turns it into a structured prompt with
sections for intent, context, style, constraints, output format, and revision
checks.

## What it adds

- Clear writing objective
- Genre, tone, and length guidance
- Scene and character continuity notes
- Revision checklist for consistency
- Output format that is easy to paste into an AI writing session

## Notes

This is an independent companion utility for organizing writing prompts. It is
not an official AI service, model, or model download.
